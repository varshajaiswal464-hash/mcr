## 🚀 How It Works
📖 Welcome to Marwari College is the front page of this project. It contains four main buttons:
🏠 Home – Displays information and an introduction about Marwari College.
🔐 Login – Allows users to enter their username and password to access the system.
📝 Register – Enables students to fill in their details and create a new registration.
📍 Contact Info – Shows the location and contact information of Marwari College.
