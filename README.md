## 🏫 About Marwari College Ranchi
- This project is a simple and informative college website developed to provide basic information about Marwari College, Ranchi. 🎓
- The website includes a welcoming homepage with navigation buttons such as 🏠 Home, 🔐 Login, 📝 Register, and 📍 Contact Info.

## 🚀 Features
-  🏠 Home Section displaying information about Marwari College, Ranchi.
-  🔐 Login Page for user authentication with Username and Password fields.
-  📝 Registration Page for new student registration.
-  📍 Contact Information Page with college location and details.
- 🎨 User-Friendly Interface with simple and clean design.
-  🧭 Easy Navigation using Home, Login, Register, and Contact Info buttons.
- ⚡ Fast and lightweight web application built using HTML, CSS, and JavaScript.
## 💻 Technologies Used
-  🌐 HTML 
-  🎨 CSS  
-  ⚡ JavaScript  
## ⚙️ How It Works
- 🚀 When the website opens, users are welcomed with a "Welcome to Marwari College" page.
-  🏠 Home Button – Displays information about Marwari College, Ranchi.
-  🔐 Login Button – Allows users to enter their Username and Password.
-  📝 Register Button – Provides a student registration form for new users.
-  📍 Contact Info Button – Shows the college location and contact information.
  ## 🚀 Future Enhancements
-  📚 Course Information Section – Provide detailed information about courses, fees, and eligibility criteria.
-  📝 Online Admission Form – Allow students to apply for admission directly through the website.
-  📢 Notice Board – Display important announcements, notices, and updates.
-  📅 Event Management – Showcase upcoming college events, seminars, and programs.
-  🏆 Placement & Achievements Section – Highlight placement records and college achievements.
